# poker-test

A Clojure application to determine poker hands

## Usage

Build the code with:

```lein uberjar```

Then it with:

```cat ./clojurecodingexercise/poker-hands.txt | java -jar target/my-poker-solution.jar```

Run the tests with:

```lein test```
